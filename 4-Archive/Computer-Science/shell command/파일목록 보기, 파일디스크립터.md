### 파일 목록 보기(파일디스크립터 확인 위해)

```
- yum install lsof
  - 설치
- lsof -p <PID> | wc -l
  - 파일 디스크립터 갯수 확인

열린포트 확인
sudo lsof -PiTCP -sTCP:LISTEN

특정 포트 확인
`sudo lsof -i :3000`
```
